A propriedade de exibição

display: inline

display: block é uma propriedade CSS que faz um elemento ocupar toda a largura disponível em sua linha, iniciar em uma nova linha e empurrar outros elementos para baixo CSS linha
Exemplos comuns de tags que são block por padrão são (div), (p), e (h1).

display: inline-block

display: inline-block combina características de display: inline e display: block. Ele permite que os elementos fiquem na mesma linha uns com os outros (como os inline), mas também possibilita definir larguras, alturas e margens (como os block). características combina
